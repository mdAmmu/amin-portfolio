# Product Images

Store product images in this directory. 

## Recommended Specifications

- **Format**: SVG, PNG, or JPG
- **Aspect Ratio**: 16:9 (recommended for best display)
- **Dimensions**: Minimum 1200x675px for high-quality display
- **File Size**: Optimize images to keep under 500KB for fast loading

## Naming Convention

Use kebab-case and descriptive names:
- `dashboard-ui-kit.svg`
- `ai-chat-component.png`
- `landing-page-template.jpg`

## Image Best Practices

1. **High Quality**: Use crisp, professional images
2. **Consistent Style**: Maintain visual consistency across products
3. **Showcases Value**: Show the product in action or highlight key features
4. **Optimized**: Compress images without losing quality
5. **Accessible**: Use meaningful alt text in the MDX frontmatter
