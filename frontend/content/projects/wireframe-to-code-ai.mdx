## Project Overview

Wireframe to Code AI is an innovative platform designed to revolutionize the way developers and designers iterate on web layouts. Traditionally, converting a design sketch into code is a manual, time-consuming process. This project automates that workflow by using AI to "see" a wireframe and generate the corresponding semantic HTML and Tailwind CSS logic.

Built with a modern stack including Next.js 14 and Neon PostgreSQL, it provides a seamless interface for uploading wireframe images, describing specific requirements, and receiving high-quality, editable code in real-time.

## Key Features

- **AI-Powered Vision**: Analyzes uploaded wireframe images using GPT-4o's vision capabilities to identify layout structures, components, and design patterns.
- **Instant Code Generation**: Generates clean, responsive React code with Tailwind CSS within seconds of uploading a design.
- **Interactive Code Editor**: Uses Sandpack to provide a live-previewing code editor for immediate experimentation with the generated output.
- **Project Persistence**: Saves generated designs and code snippets to a PostgreSQL database for team collaboration and future reference.
- **Cloud Image Storage**: Integrates with Cloudinary for secure and optimized management of wireframe assets.

## Technical Implementation

### Architecture

The system follows a full-stack Next.js architecture, utilizing the App Router for efficient server-side rendering and API management.

- **Frontend**: Scalable React components using Radix UI and Tailwind CSS.
- **Persistence**: Neon PostgreSQL with Drizzle ORM for type-safe database interactions.
- **AI Integration**: OpenAI SDK for vision processing and streaming chat completions.
- **Asset Management**: Cloudinary for image hosting and optimization.

### Frontend

The frontend focuses on responsiveness and a premium developer experience. We use Lucide-React for consistent iconography and Radix UI primitives for accessible components like dialogs and tooltips.


### Backend & AI

The backend is built using Next.js Route Handlers. It manages user requests, database migrations, and orchestrates the AI logic.

```typescript
// app/api/wireframe-to-code/route.tsx snippet
export async function POST(req: NextRequest) {
  const { description, imageUrl, model, uid, email } = await req.json();

  const result = await db.insert(WireframeToCodeTable).values({
    uid,
    description,
    imageUrl,
    model: model || 'gpt-4o',
    createBy: email,
  }).returning({ id: WireframeToCodeTable.id });

  return NextResponse.json(result);
}
```

**AI Pipeline:**
- **Vision Model**: GPT-4o identifies spatial relationships and component types from the input image.
- **System Prompting**: We provide detailed instructions to the model to ensure it outputs valid React/Tailwind code without unnecessary boilerplate.

## Challenges & Solutions

### Challenge 1: Layout Accuracy

**Problem**: Initial versions often struggled with complex grid layouts or overlapping elements in hand-drawn sketches.

**Solution**: We refined the prompt engineering strategy to include "visual cues" and explicitly requested a responsive grid system (Tailwind Grid) for the output, which significantly improved the structural accuracy of the generated code.

### Challenge 2: Real-time Feedback

**Problem**: Large code generations could take several seconds, leading to a poor user experience during "wait" times.

**Solution**: Implemented streaming responses from the OpenAI API, allowing the user to see the code being "typed" in real-time, which creates a much more engaging and immediate feedback loop.

## Results & Impact

- **Speed**: Reduced initial prototyping time from 2-3 hours to under 30 seconds.
- **Accuracy**: achieving ~85% layout match for clear, high-contrast wireframes.
- **Developer Experience**: Integrated a live preview using Sandpack, enabling zero-config testing of generated code.

## Key Learnings

1. **AI Vision Limits**: Learned how to optimize image contrast and resolution to improve model interpretation.
2. **Type-Safe DB**: Leveraged Drizzle ORM to maintain strict typing between the Next.js API and PostgreSQL, preventing runtime database errors.
3. **UX for AI**: Discovered that streaming UI is critical for AI-driven applications to maintain user trust.
